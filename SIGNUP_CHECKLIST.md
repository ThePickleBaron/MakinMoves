## Priority 2: Content/Affiliate (Passive Income)

### Hosting & Site Building
- [ ] **Ghost** (recommended blogging platform)
  - URL: https://ghost.org
  - Cost: $29–$199/mo (start with basic tier)
  - Why: Great for SEO, affiliate-friendly, newsletter built-in

- [ ] **Netlify** (free static hosting, alternative)
  - URL: https://netlify.com
  - Cost: Free tier available
  - Why: Cheap/free for Astro sites

### Affiliate Networks (Sign Up to Get Commissions)
- [ ] **Amazon Associates**
  - URL: https://affiliate-program.amazon.com
  - Why: Highest traffic, 3–10% commissions
  - What you need: Email, address, tax ID (SSN)

- [ ] **ShareASale** (multi-brand affiliate network)
  - URL: https://www.shareasale.com
  - Why: 1000+ merchant programs, 15–30% commissions
  - What you need: Email, website URL, tax info

- [ ] **Impact** (formerly Impact Radius)
  - URL: https://impact.com
  - Why: Premium brands, higher commissions
  - What you need: Email, website

### Ad Networks
- [ ] **Google AdSense**
  - URL: https://adsense.google.com
  - Why: Most popular, $100 minimum payout
  - What you need: Email, website, tax info

- [ ] **Mediavine** (higher-paying alternative)
  - URL: https://www.mediavine.com
  - Cost: Requires 25k+ monthly traffic
  - Why: 75/25 split (you get 75%), higher CPMs

## Safety Reminders

⚠️ **NEVER:**
- Share your Stripe API keys publicly
- Store passwords in plain text files
- Use same password across services
- Enable 2FA without saving backup codes

✅ **ALWAYS:**
- Use a password manager (1Password, Bitwarden, LastPass)
- Enable 2FA on Stripe, Binance, email
- Keep backup codes in a safe place
- Use unique, strong passwords
