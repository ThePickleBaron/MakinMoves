## HOSTING & PUBLISHING SETUP

**Platform Options:**
1. **Ghost** (Blog + SEO optimized): $9/month
2. **WordPress + Astra theme**: $8-15/month + theme
3. **Webflow**: $12-165/month (overkill for affiliate sites)
4. **Substack**: Free (limited SEO)

**Recommendation:** Ghost ($9/month per site = $27/month for 3 sites)

**Setup Steps:**
1. Register 3 domains (Namecheap): $10.98 × 3 = $33
2. Set up Ghost on each domain: Free account, link domain
3. Install SEO plugin (built-in Ghost)
4. Create pages (About, Privacy Policy, Affiliate Disclosure)
5. Publish 5 articles Week 1 (3 on Site 1 Email, 1 on Site 2 AI, 1 on Site 3 SaaS)
6. Publish 10 articles Week 2-3 (continue spreading across sites)
7. Set up email capture (Ghost built-in, or ConvertKit if selling course later)
