## INFRASTRUCTURE REQUIREMENTS

### One-Time Setup (Jason + Claude, 4–6 hours total)
1. **Substack paid tier:** 30 min (add bank + tax info)
2. **Stripe account:** 20 min (verify identity)
3. **Gumroad account:** 15 min (connect Stripe)
4. **SendGrid API key:** 10 min (generate key, store in .env)
5. **Zapier automation:** 30 min (set up weekly briefing email + Substack delivery)
   - **Alternative:** GitHub Actions cron job (requires CloudFlare worker or equivalent for Substack API integration; more complex)
6. **Google Sheets tracker:** 15 min (revenue + subscriber tracking)

**Automation note:** Substack doesn't have a native REST API for paid content publication. Options:
- **Option A:** Publish manually (15 min/week, Jason-compatible)
- **Option B:** Use Zapier to trigger email sequence (30 min one-time setup, fully automated)
- **Option C:** Build custom script with Playwright to automate Substack login + publish (2 hours, higher maintenance risk)

**Recommendation:** Manual publish + Zapier email delivery. Lowest maintenance, fits Jason's 2–3 hr/week constraint.

### Recurring Infrastructure Monitoring
- **Weekly:** Check Substack publish succeeded + email delivered
- **Monthly:** Verify Gumroad payment processing + customer list growth
- **Monthly:** Verify SendGrid delivery rates (spam complaints, bounce rate)
