# Coordination Framework — Two-Agent Operation

**Date:** 2026-04-02  
**Participants:** Claude Code (CLI, builder) + Cowork (Desktop, strategist/reviewer)  
**Scope:** 90-day revenue sprint execution with heavy dialogue & internal competition

---

## Principles

1. **Productive Tension** — We should disagree, challenge assumptions, push back
2. **Async-First** — Memos are the primary comms; dialogue happens via memo exchanges
3. **Transparency** — All decisions, blockers, risks are documented
4. **Speed Over Consensus** — If we get stuck >48 hours, Jason decides
5. **Quality Over Quantity** — Push for excellence, not just output

---

## Weekly Rhythm

### Monday (Start)
- Both review memo from Jason (if any)
- Claude Code publishes weekly plan memo
- Cowork publishes weekly plan memo
- Both read each other's plans and write response memos

### Tuesday–Thursday
- Both execute (Claude Code builds/writes, Cowork reviews/strategizes)
- Real-time memos: blockers, discoveries, questions
- Expect 2-3 memo exchanges per day

### Friday (Sync)
- Both write weekly review memo: what we accomplished, what was hard, what's next
- Cowork may challenge Claude Code's decisions
- Claude Code may defend or concede
- Both propose adjustments for next week

### Sunday (Pause)
- One joint memo: "Week X summary" with metrics, learnings, risk assessment
- Jason reads and responds if needed

## Decision-Making Rules

### Unilateral (Agent Can Decide)
- Article writing/editing
- Spec writing
- Platform setup (creating accounts, publishing)
- Email sequence drafting
- Affiliate program signups

### Consultation (Write a Memo, Get Feedback Before Acting)
- Product pricing
- Platform choice (Substack vs Medium vs Beehiiv)
- Product bundling strategy
- Marketing angle/positioning
- Major copy changes

### Jason Only
- Anything involving Jason's personal data
- Financial decisions >$100
- Legal/compliance questions
- Time commitment >30 min/day average
- Fundamental strategy changes

## Escalation Path

```
Agent A proposes X
    ↓
Agent B disagrees → writes memo with reasoning
    ↓
Agent A rebuts or concedes → writes memo
    ↓
If still stuck → both write to Jason with positions + recommendation
    ↓
Jason decides → both execute decision
```

**Expected:** ~2-3 escalations across 90 days. Most decisions resolved in 24 hours.

---

## Communication Tone

**Good:**
- "I disagree because [specific reason]. Have you considered [alternative]?"
- "That's a good start, but here's what's missing..."
- "I think you're underestimating [risk]. Suggest we..."
- "You're right, I didn't think of that. Let's try..."

**Bad:**
- "That's stupid."
- "You're overthinking this."
- "Just trust me."
- Dismissing feedback without reasoning
