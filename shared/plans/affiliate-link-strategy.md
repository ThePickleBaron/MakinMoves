## TRACKING SETUP

### What to Track

1. **Clicks per link** — Which articles drive most traffic?
2. **Conversions per link** — Which affiliate programs convert best?
3. **Revenue per link** — Which programs are most lucrative?
4. **Email conversions** — Which emails drive upsells?
5. **Product sales** — Gumroad + Etsy weekly revenue

### Tools

**Option A: Manual Spreadsheet**
- Date | Link | Source | Clicks | Conversions | Revenue
- Update weekly
- Simple but time-consuming

**Option B: UTM Parameters**
- Add to each link: `?utm_source=substack&utm_medium=email&utm_campaign=article4`
- Use Google Analytics to track
- More automated, requires GA setup

**Option C: Notion Tracker**
- Database: Links, articles, programs, clicks, conversions, revenue
- Manual input, but organized
- Can add formulas for ROI

**Recommendation:** Start with UTM parameters + Notion tracker (hybrid)

## DISCLOSURE REQUIREMENTS

### Affiliate Disclosure
Add to top of each article with affiliate links:

```
**Affiliate Disclosure:** This article contains affiliate links. 
If you purchase through these links, I earn a commission. 
This doesn't cost you anything extra, and I only recommend 
products I genuinely use and recommend.
```

### Trust Building
- Only recommend products you use yourself
- If you don't use it, don't link it
- Be honest about pros/cons
- Disclose commission amounts (e.g., "50% recurring commission from Sharesight")

## AFFILIATE ETHICS

**Do's:**
- Link to tools you genuinely use
- Disclose affiliate relationship
- Provide honest reviews (pros + cons)
- Only promote products that add value
- Track results and optimize

**Don'ts:**
- Link to tools you haven't tested
- Hide affiliate relationship
- Recommend based on commission alone
- Spam affiliate links in every article
- Violate platform terms (FTC guidelines)
