## TRACKING SETUP

### What to Track

1. **Clicks per link** — Which articles drive most traffic?
2. **Conversions per link** — Which affiliate programs convert best?
3. **Revenue per link** — Which programs are most lucrative?
4. **Email conversions** — Which emails drive upsells?
5. **Product sales** — Gumroad + Etsy weekly revenue

### Tools

**Option A: Manual Spreadsheet**
- Date | Link | Source | Clicks | Conversions | Revenue
- Update weekly
- Simple but time-consuming

**Option B: UTM Parameters**
- Add to each link: `?utm_source=substack&utm_medium=email&utm_campaign=article4`
- Use Google Analytics to track
- More automated, requires GA setup

**Option C: Notion Tracker**
- Database: Links, articles, programs, clicks, conversions, revenue
- Manual input, but organized
- Can add formulas for ROI

**Recommendation:** Start with UTM parameters + Notion tracker (hybrid)
