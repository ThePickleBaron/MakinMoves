## THREAD 2: Technical + Fundamental Mashup

### Tweet 1:
```
Most traders split into camps: "technicians" vs. "fundamentalists."

False dichotomy.

Technical = what the market IS doing
Fundamental = what the market SHOULD be doing

The divergence between these two is where conviction trades are born.
```

### Tweet 2:
```
Example: Housing thesis I'm short.

Fundamentals: Supply exhausted, construction starts down 74%, BLS rents at ATH. Market mispriced for relief.

Technicals: XHB broke support at $78, IYR broke $52. Volume accelerating on breakdown.

Both agree. Conviction increases.
```

### Tweet 3:
```
Where technicals and fundamentals DISAGREE = risk.

Fundamentals say "short" but technicals show support holding, weak breakdown volume = thesis under pressure.

Could be early. Could be wrong.

Size according to thesis conviction, not technical signals alone.
```

### Tweet 4:
```
The framework:

1. Start with thesis (macro data + divergence)
2. Confirm with technicals (price/volume)
3. Both agree = higher conviction, bigger size
4. They disagree = lower conviction, smaller size or wait
5. Stops at thesis break level (not technical support)

Thesis > technicals. Always.
```

### Tweet 5:
```
Your thesis is your conviction.
Your technicals are confirmation.
Your stops are your thesis break level.

When thesis breaks, exit. Don't wait for technicals to confirm a loss.

Discipline > prediction.
```

## THREAD 3: The Cascade Effect

### Tweet 1:
```
Institutional money moves slow. Market prices slow.

But once the cascade starts, it accelerates.

Here's how divergence cascades from first movers to herd:
```

### Tweet 2:
```
Stage 1: Divergence (you see it, few others do)
- Thesis clear to independent thinkers
- Price hasn't moved
- Size: Medium conviction, measured entry

Stage 2: Early adoption (smart money wakes up)
- Volume picks up, price moves 10-20%
- Size: Increase if thesis intact
```

### Tweet 3:
```
Stage 3: Acceleration (FOMO hits)
- Everyone's watching, volume spikes
- Price moves 30-50%
- Don't add here. Technicals confirm but thesis is now consensus.

Stage 4: Normalization
- What was "contrarian" is now "obvious"
- Exit: 60-80% of position. Let 20-40% ride.
```

### Tweet 4:
```
Key to cascade trading:

1. Enter at Stage 1 (30-40% of conviction size)
2. Add at Stage 2 if thesis intact (30-40% more)
3. Hold at Stage 3 (don't FOMO add)
4. Exit at Stage 4 (thesis = market's new floor)

You don't need the exact top. Exit before the cascade exhausts.
```

### Tweet 5:
```
Most traders miss cascades because they:

- Wait for "confirmation" (Stage 3, 70% of move gone)
- Don't have thesis conviction (can't tell Stage 1 from noise)
- Can't hold through doubt (exit at Stage 2 volatility)

Start with thesis. Trust the data. Let time prove you right.
```

## THREAD 4: Conviction Structuring

### Tweet 1:
```
How do you size a conviction trade?

Not by portfolio allocation %. Not by Kelly Criterion. Not by some risk management formula.

By thesis conviction.

Here's my framework:
```

### Tweet 2:
```
Level 1 (Hypothesis):
- Data suggestive, not conclusive
- 1-2 real counter-arguments
- Size: 5-10% of conviction allocation
- Stop: Thesis break level

Level 2 (Thesis):
- Data clear, counter-arguments weak
- Size: 20-30%
- Stop: Thesis break level
```

### Tweet 3:
```
Level 3 (Cascade):
- Data overwhelming, thesis battle-tested
- Held through volatility, thesis intact
- Size: 40-50% of conviction allocation
- Stop: Thesis break level

Your conviction should match your position size. Not the other way around.
```

### Tweet 4:
```
Most traders do it backwards.

They size first, then convince themselves of thesis.

"Kind of sure" = small position
"Very sure" = medium position
"Absolutely certain (thesis tested, data clear)" = large position

Conviction drives size. Size doesn't create conviction.
```

### Tweet 5:
```
Conviction structuring:

1. Thesis conviction = position size
2. Thesis break = automatic exit
3. Time in trade = until thesis converges to consensus
4. Profit target = let it run until thesis is obvious

You're trading the gap between what you know and what the market prices. When that gap closes, you're done.
```
