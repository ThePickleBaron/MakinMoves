## Workflow Summary

### Daily (During standup prep, 15:00–16:00)

1. Open `daily_tracking_YYYY-MM-DD.md` (today's log)
2. Pull revenue from all platforms
3. Update cumulative vs. scenarios
4. Check if any decision triggers crossed
5. If trigger crossed, open `decision_tree_revenue_actions.md`
6. Follow the action for that trigger
7. Draft standup section
8. Escalate to CEO if needed

### Weekly (Friday 18:00)

1. Compile week's performance (daily logs)
2. Update `cac_ltv_analysis.md` with real data
3. Populate `weekly_summary_2026-04-DD.md` template
4. Post to `CFO/outbox/`
5. Note any items for CEO review

### Checkpoints (Day 7, 14, 21)

1. Compare actual cumulative revenue to all 3 scenario targets
2. Determine which scenario we're tracking
3. Follow decision tree for that scenario
4. Escalate to CEO with recommendation
