- Defense: Choose ONE niche (writing, B2B, etc.) + add community (Discord, updates)

- Freelance tools niche: Winnable (no dominant player, lower competition)

### 5. Success Metrics (April 30 Targets)

**P1:** 20+ total sales (across all 3 variants, proves product works)
- If <5: Pause P1, focus on P2/P3
- If 5–10: One variant is working, scale that one
- If 20+: All variants viable, pick winner by conversion rate

**P2:** 5+ responses from 10 prospects (40%+ response rate)
- If <50% response: Expand prospect list to 30+, different channels
- If 50–75%: Positioning works, book discovery calls
- If 75%+: Both angles resonating, segment customers (some want generalist, some specialist)

**P3:** 100–300 visitors (both sites combined) + 3–5 backlinks earned
- If <100: Niche may be wrong OR SEO timeline is longer than expected
- If 100–200: On track, need more content/backlinks
- If 200–300: Strong signal, continue niche, expand to 50+ articles by Month 3
