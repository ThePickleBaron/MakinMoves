## Top 3 Affiliate Networks (Primary + Secondary)

### 1. Shareasale
- **Commission structure:** Per-affiliate program (varies 5–50%)
- **Payout:** 30 days, $100 minimum
- **Best for:** High-ticket B2B tools, SaaS
- **Programs available:** 1000+
- **Our use:** ConvertKit, Monday.com, Asana, Jasper, etc.

### 2. Impact (formerly Impact.com)
- **Commission structure:** Per-affiliate program (varies 10–40%)
- **Payout:** Net 30, $50 minimum
- **Best for:** Enterprise SaaS, high-value partnerships
- **Programs available:** 500+
- **Our use:** Stripe, PayPal, Zapier alternatives

### 3. CJ Affiliate (Commission Junction)
- **Commission structure:** Per-affiliate program (varies 2–25%)
- **Payout:** Monthly, $25 minimum
- **Best for:** Broad coverage, consumer tools
- **Programs available:** 3000+
- **Our use:** Software tools, hosting, domain registrars
