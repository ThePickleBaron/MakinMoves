## USING THIS LIBRARY WEEKLY

### Sunday Evening Routine (30 min)

1. **Open WEEKLY_RISK_CHECKLIST.md** (5 min)
   - Fill in metrics for each stream
   - Mark status (🟢🟡🔴)
   - List any red flags

2. **Check DEPENDENCY_MAP_VISUAL.md dashboard** (3 min)
   - See compound risk level
   - Confirm no emerging patterns

3. **Make decisions if RED flags** (5-10 min)
   - Refer to decision matrix (Part 9 or Checklist)
   - Document decision

4. **Log results** (2 min)
   - Copy checklist to `CEO/logs/weekly-risk-log.md`
   - Include week number, status, any decisions made

5. **Schedule follow-up** (2 min)
   - If YELLOW flags → Plan fix for next week
   - If RED flags → Schedule emergency check-in (24 hours)

## DURING CRISIS (When RED Flag Detected)

### 4-Hour Response Window

**Hour 0-1: Diagnosis**
- Confirm the problem is real
- Find right section in CONTINGENCY_ACTIVATION_PLAYBOOK.md
- Check pre-requisites (is backup ready?)

**Hour 1-3: Execution**
- Follow step-by-step instructions in playbook
- Activate backup platform / hire / descope
- Monitor first 1-2 transactions

**Hour 3-4: Documentation & Communication**
- Email team: Brief summary of action taken
- Document decision in `board/decisions/decision_YYYY-MM-DD_[issue].md`
- Log to weekly risk log
