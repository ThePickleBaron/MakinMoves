**Task 1.1: Build Substack Target List (45 min)**
- Go to: `substack.com`
- Search keywords: "AI", "SaaS", "productivity", "tools", "automation"

- Columns: First Name | Substack URL | Recent Topic | Email (if visible) | Subscriber Count | Engagement Notes

- **Tool:** Spreadsheet (Google Sheets or Excel)
- **Check-off:** ✓ Done when you have 50+ writers with contact info

## CRITICAL HANDOFF NOTES FOR COO

**If running ahead of schedule:**
- Start Week 2 tasks early (expand to 20+ partnerships)
- Reach out to non-responders from Week 1 (higher conversion on follow-up)
- Create additional partner tiers (micro-influencers, niche communities)

**If running behind schedule:**
- Focus on high-ROI channels only (Substack, Twitter, LinkedIn)
- Deprioritize slow channels (Medium, Notion Creators, Agencies)
- Lean on affiliate networks (passive, self-serve recruitment)

**Watch-outs:**
- Spam folder: Check spam/promotions folder daily—many partner responses land there
- Response rate expectation: Assume 10–20% response rate on cold outreach (1 in 5–10 partners interested)
- Time investment: This is the heaviest lift—Week 2+ require much less daily effort (passive partner revenue)

**Red flags to report to CEO:**
- If <5 partnerships signed by end of Week 1, re-evaluate messaging/targeting
- If conversion rate <1% on affiliate traffic, review product/pricing
- If top partners deliver <$100 revenue, consider deprioritizing that channel
