## DETAILED TASK BREAKDOWN BY STREAM

### DIGITAL PRODUCTS: Detailed Task List (Days 22–60)

#### Template Creation Template (Use This for Every Template)
**Time per template:** 3–5 hours (end-to-end)
1. Choose niche/template type (15 min)
   - Example: "SaaS Founder Financial Dashboard"
   - Research: Is this niche in demand? Will people pay for this?
2. Create in Canva or Notion (2–3 hours)
   - Use Canva Pro template library for speed
   - OR create from scratch in Notion with sample data
   - Include 10–15 customizable sections
3. Write product description (20–30 min)
   - Headline: "Complete [Niche] Operating System for [Audience]"
   - Bullet points: 5 key benefits
   - Include use cases: 3–4 examples
4. Create preview image/GIF (20 min)
   - Canva export: 1600×900px JPG
   - Notion: Screenshot of completed template
5. List on platform (10 min)
   - Gumroad: Set price, write description, upload preview
   - Etsy (if Canva): Same process
6. Promote on social (15 min)
   - Reddit: Post in r/Notion, r/[niche], r/SideHustles
   - Twitter: Tweet preview + link

#### Digital Products Week-by-Week Goals
**Week 4:**
- [ ] Template #2 (if Phase 1 proved niche) or NEW niche template #1 (if pivot)
- [ ] Email: Export customer email list from Gumroad if any sales
- [ ] Social: Post 3–4 times on Twitter + Reddit about template(s)

**Week 5:**
- [ ] Template #3 (if Week 4 saw 3+ sales)
- [ ] OR Premium bundle: Package templates #1–2 at $99 (test premium pricing)
- [ ] Email: Send email to past buyers: "Template #3 just launched!" (1–3 sales expected)

**Week 6:**
- [ ] Bundle launch finalized (or Template #4)
- [ ] Update existing templates based on customer feedback (30 min)
- [ ] Social: Increase posting to 3–4x/week

**Week 7–8:**
- [ ] Maintain 4–5 templates; minimal new creation unless revenue >$30/week
- [ ] Email: Send promotional email 1x/week to buyers

**Niche Backup Plan (If Digital Revenue = $0 by Day 24):**
- Backup niche from Phase 1 research already identified
- 48-hour turnaround: Create 1 template in new niche (Days 24–26)
- Test on same platforms; evaluate by Day 31

### PRINT-ON-DEMAND: Detailed Task List (Days 22–60)

#### Design Creation Template (Use This for Every Design)
**Time per design:** 30–45 minutes
1. Design in Canva (20–30 min)
   - Use Canva Pro design library (speed over perfection)
   - Focus: Clear message, readable from thumbnail, niche-specific
   - Size: 1080×1080 minimum for all platforms
2. Export (5 min)
   - Canva → PNG (transparent background)
   - Batch export to folder
3. Upload to platforms (10 min)
   - Redbubble: Select product types (shirt, hoodie, mug)
   - Merch by Amazon: If approved (takes 1–2 weeks)
   - Printful (optional): If Redbubble saturates
4. Write product description (5 min)
   - Title: "[Niche] [Humor/Message] T-Shirt"
   - Tags: 5–10 niche keywords

#### Design Production Velocity
**Week 4:** 5 designs (target total 10–15)
**Week 5:** 5 designs (target total 18–23)
**Week 6:** 5 designs (target total 24–29)
**Week 7:** 5 designs (target total 29–34; adjust based on performance)
**Week 8:** Final batch (target total 30)

**If production <5 designs/week:** Increase priority; block 2–3 hours/day for design batch

#### Design Focus Strategy
**Week 4–5:** Create designs across 3–4 different themes
- Examples: Developer humor, ADHD tools, SaaS founder memes, etc.
- Goal: Find 2–3 themes that get clicks/sales
**Week 6–8:** Replicate winners
- If "Developer ADHD T-Shirt" gets 15+ clicks: Create 3 color variations + related designs
- If "SaaS Founder Guide" t-shirt sells 2+: Create 3 new designs around that theme

#### Merch by Amazon Strategy (If Approved)
- Merch approval takes 1–3 weeks
- Meantime: Build portfolio on Redbubble (lower barrier, faster feedback)
- Once Merch approved: Upload all 10+ approved designs immediately
- Merch algorithm favors new designs; expect 30–50% of sales from Merch within 2 weeks of upload

#### Design Quality Checklist
- [ ] Readable at 200×200px (thumbnail size)
- [ ] Message clear in under 3 seconds
- [ ] Design unique (not generic)
- [ ] Niche-specific (not universal)
- [ ] Colors test well on multiple products (white shirt, dark shirt, mug)

### AFFILIATE MARKETING: Detailed Task List (Days 22–60)

#### Article Writing Template (Use This for Every Article)
**Time per article:** 2–3 hours (1,200–1,500 words)
1. Outline (15 min)
   - Target keyword research (Google Trends, Ahrefs if available)
   - Example keyword: "Best AI writing tools 2026"
   - Outline: 5–7 sections with h2 headers
2. Write (1–1.5 hours)
   - Intro: Hook + keyword mention
   - 5–7 body sections with examples
   - Conclusion: CTA to email list or affiliate link
3. Edit + optimize (20–30 min)
   - Include internal links to previous articles
   - Add 2–3 affiliate links (naturally, not forced)
   - Optimize meta description (150 chars with keyword)
4. Format + publish (15 min)
   - WordPress: Add featured image, set category, schedule
   - Yoast SEO: Optimize for keyword (aim for Green light)
5. Promote (15 min)
   - Twitter: Post link + teaser
   - Reddit: Post in relevant subreddits (2–3 communities)
   - Email: Add to email queue for next newsletter

#### Article Calendar for Phase 2 (Weeks 4–8)

**Theme: Best AI/SaaS Tools 2026**

| Week | Articles | Titles |

| **Week 4** | #6–7 | "Best AI Writing Tools for Content Creators" / "Best Project Management SaaS for Teams" |

| **Week 6** | #10–11 | "Free AI Tools for Freelancers" / "Best Email Automation Tools 2026" |

- Week 6+: Tackle higher-volume keywords (more traffic, more affiliate opportunity)

### If Revenue Drops 50% Week-over-Week
```
TRIGGER: Weekly_Revenue_Change < -50%

ACTION:
1. STOP: Do not continue executing as-is
2. DIAGNOSE: What changed? (New competitor? Platform algorithm shift? Email unsubscribes?)
3. ESCALATE: Write immediate decision request to CEO inbox
4. Options to evaluate:
   a. Increase marketing spend (if revenue > $50 but traffic dropped)
   b. Pivot niche (if revenue < $10 and hasn't improved)
   c. Improve product quality (if traffic high but conversion low)
5. Wait for CEO approval before major changes

APPROVAL: CEO makes strategic decision; COO executes
```
