## QA Criteria (Every Prompt Must Pass All 3)
1. **Specificity test** — produces a concrete, usable output on first run (not vague guidance)
2. **Speed test** — setup time under 10 seconds; copy-paste and go
3. **Edit rate test** — output requires less than 20% editing to use professionally
4. **Voice test** — includes a 1-sentence "why this works" written in human, non-AI-sounding language

## Product Structure (PDF Layout)

1. **Cover page** — Title, tagline: "50 tested prompts for every stage of your freelance business"
2. **How to Use This Pack** (1 page) — 3 rules: customize the brackets, test once before client use, save your best outputs
3. **Section 1–5** — Each section opens with a 1-paragraph buyer pain statement, then prompts
4. **Each prompt page format:**
   - Prompt title
   - "When to use this" (1 sentence)
   - The prompt (formatted, bracket placeholders in **bold**)
   - "Why it works" (1–2 sentences, human voice)
   - Sample output excerpt (3–5 lines in gray box)
5. **Bonus: Prompt Customization Cheatsheet** (1 page) — how to modify bracket variables for any niche
