## QUICK REFERENCE: KEY DECISIONS

### 1. What's the Product Mix Strategy?
- **Digital Products:** 25% effort → 4–5 templates, $50–$220 revenue by Day 60
- **Print-on-Demand:** 30% effort → 25–30 designs, $240–$640 revenue by Day 60
- **Affiliate Marketing:** 45% effort → 14–15 articles, 200+ email subs, $0 revenue (long-play)

**Decision rule:** If any stream $0 by Day 24, execute niche pivot (48-hour test). Reallocation based on Week 2 winners.

### 2. What Channels Should We Use?
- **Digital Products:** Reddit + Twitter + Gumroad discovery
- **PoD:** Reddit niches + Twitter + Merch by Amazon (once approved)
- **Affiliate:** Content (SEO) + Email newsletter + Reddit/Twitter engagement

**Decision rule:** No paid ads in Phase 2. All organic. Paid ads only Phase 3+ if revenue validates.

### 3. What Pricing Should We Use?
- **Digital Products:** $29 entry-level, $39–$79 mid-market, $99–$199 premium (test only after 10+ sales)
- **PoD:** Platform defaults ($16.99 base); test $19.99 on bestsellers Week 6+
- **Affiliate:** Commission-based (no control); target 20–40% recurring SaaS programs

### 4. What Revenue Targets Should We Hit?
- **By Day 28:** $40–$105
- **By Day 42:** $70–$190
- **By Day 60:** $200–$600 cumulative

**Success threshold:** $200+ = Phase 3 approved. $100–$200 = Extend Phase 2. $0–$100 = Strategy reset needed.

### 5. What If Phase 1 Failed?
- **If Digital $0 by Day 24:** Pivot niche (3-day test of backup niche)
- **If PoD $0 by Day 28:** Increase production to 2 designs/day (volume solves discovery)
- **If Affiliate $0 by Day 60:** Expected. Don't panic. Affiliate takes 8–12 weeks. Focus on email list growth.

## Q&A: COMMON STRATEGY QUESTIONS

**Q: Why 45% Affiliate if we won't see revenue until Day 90?**
A: Affiliate is your long-term passive play. Phase 2 builds the infrastructure (email list, content authority, SEO foundation) that pays dividends starting Week 12+. Phase 3–4 focus is affiliate monetization. Can't afford to skip this foundational work.

**Q: What if PoD doesn't hit $240? Should we pivot?**
A: Not necessarily. PoD is volume-dependent. If 25+ designs are live but only 30 sales (vs. target 40), it's usually a niche issue, not an execution issue. By Day 56, identify best-performing themes and double down (create variations). If <15 designs by Day 35, increase production velocity.

**Q: Can we run paid ads to accelerate?**
A: Not in Phase 2. Paid ads need minimum $100+/week spend to be meaningful. Phase 2 is proving product-market fit with organic channels. Phase 3 (if revenue validates), then test paid ads.

**Q: What happens if email list stays <100?**
A: That's your Affiliate growth signal failing. By Day 42, if <100, change strategy: Add exit-intent popup to site, create lead magnet (AI Tools PDF), guest-post outreach. If still <100 by Day 60, Affiliate timeline extends to Q3 (doesn't invalidate strategy; just delays revenue).

**Q: When do we start Phase 3?**
A: Day 61 (day after Phase 2 ends). If revenue $200+ and 2+ streams viable: Approve Phase 3 (15 min/week maintenance mode). If revenue $100–$200: Extend Phase 2 another 4 weeks. If revenue $0: Reset strategy (investigate Phase 1 assumptions).

**Q: What if the user has time post-March 31 despite planning hands-off?**
A: Systems should run fully autonomous. If user wants to intervene, they can, but it's not required. All critical decision rules are automated.
