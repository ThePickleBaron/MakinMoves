## Section 10: FAQ & Contingencies

**Q: What if Etsy shop setup is delayed (ID verification)?**
A: Skip Etsy temporarily, launch Gumroad + Redbubble first (both free). Etsy provides +20% upside but isn't critical. Return to it once verified.

**Q: What if first month revenue is $100 (pessimistic case)?**
A: Expected. Continue; pessimistic case still hits break-even by Day 100. Increase content output and lower prices to increase conversion.

**Q: Can we go live with just Gumroad + Etsy to save time?**
A: Yes, but risky. Diversification de-risks single-platform failure. Recommend all 5 within 7 days if possible.

**Q: Do we have to use all 5 platforms, or can we pick 2-3?**
A: You can pick 2-3 to launch faster, but 5+ gives better odds. Each platform has different audience, timing, and algorithm.

**Q: What if we need to pivot one stream (e.g., Redbubble not working)?**
A: Replace with similar platform (e.g., Printful, Teespring) with $0 setup. All PoD platforms are essentially free, so portfolio easily adjustable.

**Q: Should we spend the $715 reserve upfront on ads?**
A: NO. Reserve is for contingency + scaling when revenue validates. Ads before product-market fit = wasted money.

**Q: How do we handle taxes on multiple platform payments?**
A: All platforms provide 1099 documentation. Use tax-friendly accounting software (Wave, free). Current volume (< $1,000/mo) unlikely to trigger quarterly estimates, but track carefully.
