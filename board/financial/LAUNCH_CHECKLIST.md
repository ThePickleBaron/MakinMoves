## TROUBLESHOOTING

### What If No Sales by Day 7?
- This is normal. Platforms take 2-3 weeks to gain traction.
- Action: Keep publishing content. Increase quality. Check product descriptions.

### What If One Platform Gets Zero Sales?
- Expected. Not all platforms work for all products.
- Action: By Day 28, evaluate. If still zero, focus energy on other platforms.

### What If Etsy Shop Not Verified?
- Etsy may request ID verification (takes 24-48 hours).
- Action: Submit verification, continue with other platforms. Etsy can wait.

### What If Stripe Account Delayed?
- Stripe verification usually instant, but can take 1-2 days.
- Action: Set up PayPal as backup. All platforms support it.

### What If n8n Automation Fails?
- Workflows sometimes need debugging.
- Action: Test manually first (e.g., post to Twitter manually), then automate.

### What If WordPress Domain Pending?
- Domain registration takes 24-48 hours to propagate.
- Action: Use free WordPress.com subdomain temporarily, update later.
