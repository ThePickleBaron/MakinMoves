## Budget Decision Framework

**Before spending reserved funds, CFO must answer:**
1. [ ] What is the ROI?
2. [ ] Can we defer this spend?
3. [ ] Do we have revenue to justify it?
4. [ ] Is this aligned with recovery goal?

**Example decisions:**
- "Spend $50 on Pinterest ads for templates?" → Only if digital products generating $30+/week already
- "Upgrade n8n to paid plan ($50/mo)?" → Only if automation ROI is proven
- "Pay for design tool?" → Only if we can't create designs for free

---

**CFO responsibility:** Track every dollar. Report weekly to CEO on budget health vs. recovery goal.
